# OTG + BOOT 刷机/升级路径

## 已从规格书确认

- 双层 Type-A 中：
  - 上层 = `USB 2.0 OTG`
  - 下层 = `USB 3.0 Host`
- `BOOT` 键用于进入 `Maskrom / Loader`
- `升级&复位键`：短按复位，长按进入升级模式
- 支持升级方式：PC / U盘 / TF 卡

## 当前建议

### 现在只做验证，不做刷写

1. 使用数据线连接 **上层 OTG** 到电脑
2. 尝试以下两种模式：
   - 开机后普通连接，观察是否枚举设备
   - 按住 `BOOT` 再上电，观察是否枚举 Rockchip 设备
3. 若电脑识别到设备，再记录 VID/PID / 工具输出
4. 在未完成系统备份、未确认镜像匹配前，不要刷入新镜像
